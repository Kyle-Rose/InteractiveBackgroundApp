## 🧠 Project Overview  
This is a simple web application that allows users to change the page background color with the click of a button. It demonstrates **basic HTML, CSS, and JavaScript skills**, including DOM manipulation and event handling.  

This project is perfect for showcasing a beginner-friendly interactive feature for a portfolio.

---

## 🔹 Features
- **Background Color Change:** Click the button to change the page’s background color to a random color.  
- Responsive and simple design  
- Smooth color transition for a better user experience  

---

## 🛠️ Tech Stack
- **HTML5** – Page structure and content  
- **CSS3** – Styling, layout, and transitions  
- **JavaScript** – Button interactivity and DOM manipulation  
- **Netlify** – Hosting and live deployment  

---

## 📂 Project Files
- `index.html` — Main structure and content  
- `style.css` — Styling and layout  
- `script.js` — JavaScript logic for background change
